Optional parameters

Name /type

Description /example

Constraints

number

1

  • min length: 1
  • max length: 256

limit
number

Limit per page

25
  • default value: 25
  • valid values: 25, 50, 100

sort
string

Sort field name

"name"
  • default value: name
  • valid values: "name", "surname", "phone", "addedAt"

sortValue
number

Sort value (asc|desc)

1
  • default value: 1
  • valid values: -1, 1

search
string

Full text search filter (name|surname|phone|email)

""
  • default value:
  • min length: 0

string
